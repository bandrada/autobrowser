# Autobrowser

## Example Usage

**Simple same directory usage**
```bash
./switch-browser safari
./switch-browser chrome
```

**Advanced Usage**
```bash
# add to .zprofile
alias bs="~/autobrowser/switch-browser.sh safari"
alias bc="~/autobrowser/switch-browser.sh chrome"

# usage
bs
bc
```

Based on comments from https://github.com/kerma/defaultbrowser/issues/3
